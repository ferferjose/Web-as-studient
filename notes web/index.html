Tema 1: Los lenguajes de la web

  1. Introducción, evolución y estado actual de HTML.

    El HTML es el lenguaje de marcas más conocido y usado ya que es accesible, fiable, fácil de usar y de uso libre.

    Los navegadores interpretan las marcas contenidas en los documentos HTML y representan la información.
    Trabajando junto con las hojas de estilo (CSS) es un increíble lenguaje de representación de información.

  2. Estructura y sintaxis del lenguaje HTML.

    Los elementos que forman las páginas HTML están identificados por marcas o etiquetas.
    Estas marcas envuelven el contenido de la página.

    • Elementos vacíos:

      No todas las etiquetas necesitan apertura y cierre.
      No son contenedoras

    • Atributos:

      Caracterizar al elemento.
      Opcional.
      Todos los elementos pueden disponer de atributos, pero no son todos los atributos son aplicables a cualquier elemento.

    • Entidades:

      Conjunto de caracteres que permiten indicar al navegador que represente caracteres que, por diversas razones no se representan correctan.

    • Comentarios:

      Admite incluir líneas de comentarios y proporciona información a los diseñadores y programadores

    • Los elementos de un documento HTML se pueden anidar.

    • Un elemento puede contener uno o más elementos en su interior

  3. Elementos de HTML.

    Un documento HTML está compuesto de elementos organizados de manera jerárquica
    Los elementos <html>, <head>, <body> son fundamentales.

    • Etiqueta <html>:

      Elemento principal y raíz del documento HTML.
      Todo el contenido del documento está incluido en él.

    • Etiqueta <head>:

      Primera sección de la página.
      No será directamente visible en la representación del documento, ni en el navegador.
      Pero, sí tiene efecto directo y relevante en dicha representación.

      Titulo <title>:
      Se muestra en la pestaña correspondiente.
      Favorito en el navegador.
      Se tiene en cuenta para clasificar la página.


    • Etiqueta <body>:

      Representan el cuerpo de la página web
      En él, se encuentran los contenidos que se van a presentar en el navegador y la que los usuarios podrán visualizar en sus pantallas.
      Los elementos del body se clasifican en elementos de bloque y elementos de línea.

    • Normas y recomendaciones:

      Mantener una buena organización. Tabulaciones.
      Las etiquetas y los atributos no son casen sensitive.
      Dependiendo del hospedaje de la página web y del sistema operativo donde se aloja, los valores de los atributos sí que pueden ser case sensitive.
      No se puede incorporar espacios en blanco ni tabuladores en los nombres de etiquetas o atributos.
      Los atributos que toman un valor dentro de las etiquetas deben ir entrecomillados si contienen caracteres que no sean alfanuméricos.

      Los atributos que sí toman un valor alfanumérico pueden contener espacion en blanco, siempre que vayan entrecomillados también.
      HTML y los navegadores sustituyen cualquier número indefinido de espacios en blanco o de tabulaciones por un solo espacio en blanco.
      Las etiquetas de bloque (block), engloban contenido y siempre deben cerrarse. Las etiquetas de líneas(inline), no encierran contenido y pueden no cerrarse.
      Las etiquetas se pueden anidar si se desean aplicar diferentes propiedades de manera simultánea.
      Los elementos, etiqueta y atributos desconocidos serán omitidos por los navegadores.

Tema 2: Etiquetas relacionadas con texto

  1. Encabezados

    -Representan elementos de encabezado. Ej: < h1 >, hay del 1 al 6 (h1, h2, etc).

  2. Etiquetas de formato

    -Hay multitud, todas le dan cierto estilo al texto que contienen. Ej: Negrita (b), cursiva(i), cita(cite), etc.

  3. Otras etiquetas interesantes

    • -Para hacer párrafos usamos la etiqueta < p >. Se le pueden introducir más atributos como "align" que es para decir en que lugar va a estar el texto (left, center y right).
    • -Para representar un texto tal y como está en un documento HTML se usa la etiqueta < pre >.
    • -Para hacer un salto de línea se usa < br >.
    • -Para hacer una linea de separación horizontal se usa < hr >. Se puede usar también el atributo "align", el "size" que da el grosor a la línea, el "width" que da anchura y uno nuevo que se encargara de dar un aspecto tridimensional "noshade".


Tema 3: Imágenes y enlances

  1. Imágenes:

    La inclusión de imagenes simples se hace desde las primeras versiones de HTML con la etiqueta < img >.
    Importante es un elemento de línea.
    Los atributos más comunes son src, alt, with y height.
    Los tipos de ruta pueden ser relativos, absolutos y url, siendo relativa aquellas que toman en cuenta el directorio en el que se encuentra el documento; absoluto toman como base el directorio raíz del servidor y url es la dirección de la imagen en internet.

  2. Figuras:

    Una novedad en HTML son las figuras.
    Mostrar una imagen con un texto asociado.
    Para ello se utiliza la etiqueta < figure >.

  3. Enlaces:

    Para insertar enlaces se utiliza la etiqueta < a >.
    Los atributos más importantes son href, target,rel


Tema 4: Listas y viñetas

  1. Listas.

    Las listas permiten crear filas con información y marcarla con viñetas o enumeraciones.

    • Ordenadas

      Las ordenadas pueden enumerarse con números, letras o números romanos.
      Se utiliza en relaciones cuyo orden es importante.
      Sus atributos son type y start.

    • No ordenadas

      Las no ordenadas pueden usar círculos, discos o cuadrados para marcar los elementos.
      Se utiliza en realiza cuyo orden no es importante.
      Sus atributos son type y start.

    • A tener en cuenta:

      Son elementos de bloque.
      Cada elemento de la lista se incluye en un < li >.
      Se pueden anidar.
      Hay que tabular bien
